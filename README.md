# LuckyDream VN

Hệ sinh thái thời trang và du lịch thông minh tại Việt Nam.

## Tính năng chính
- **Phối đồ thông minh (AI Outfit):** Sử dụng Gemini AI để gợi ý trang phục từ ảnh người dùng tải lên.
- **Lịch trình du lịch:** Đề xuất điểm đến, ẩm thực và phương tiện di chuyển dựa trên phong cách trang phục.
- **Cộng đồng thời trang:** Chia sẻ phong cách, thuê hoặc thanh lý quần áo.
- **Chat thông minh:** Tương tác với người dùng khác và nhận tư vấn style.

## Công nghệ sử dụng
- **Frontend:** React + Vite + Tailwind CSS
- **Backend/Database:** Supabase
- **AI Engine:** Google Gemini API
- **Animations:** Motion (Framer Motion)

## Cài đặt
1. Clone repository.
2. Chạy `npm install` để cài đặt dependencies.
3. Cấu hình các biến môi trường trong file `.env` (xem `.env.example`).
4. Chạy `npm run dev` để khởi động môi trường phát triển.
